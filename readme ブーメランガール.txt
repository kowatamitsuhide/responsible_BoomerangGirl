  Visual Studio 2019 Community

・このプログラムを作ろうと思った理由
就職に向け、力をつけるためにデザインパターンを使ったプログラムを
導入したゲームを作りたいと考えたため。

・プログラムを作成する上で注意したこと
もっとわかりやすくコード量が少ないプログラムにするに場合、
どのように変更を加えればよいのかをプログラムを組みながら
試行錯誤し、コード簡略化かを図ること。

・プログラムを作成する上で大変だった所
デザインパターンを使ったプログラムがどういう動きを
するのかを考えてプログラムを組むこと。

・担当箇所
ソリューションファイル内の木幡光秀 担当箇所ございます。

・ゲーム概要
魔法がある世界で見習い魔法使いの女の子がメカに支配された森を救うために
魔法で強化したブーメランで戦う3Dブーメランアクション。

・ゲームルール
ブーメランを投げて敵を倒し、
タイムオーバーになるまで沢山敵を倒しスコア・コンボを稼ぐ

・ソリューション操作の再ターゲットというウィンドウが表示された場合

・Windows SDK:
  最新のインストールされているバージョンに設定

・プラットフォームツールセット:
  アップグレードなしを選択
